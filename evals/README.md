# Eval Suite

Benchmarks the classifier prompts generated by `agent-registry-router` across LLMs.

## Quick Start

```bash
# Install eval dependencies
make install-eval

# Copy and fill in your API keys (you only need the providers you want to test)
cp .env.example .env

# Run against all models
make eval

# Run against a single model
make eval ARGS="--models gpt-4o-mini"

# Run a specific scenario
make eval ARGS="--scenarios education_tutors --models gpt-4o-mini"

# Generate a report
make eval-report ARGS="evals/results/<your_results_file>.json"
```

## What This Tests

The library's `build_classifier_system_prompt()` generates a system prompt from a
registry of agent descriptions. This eval measures how well that prompt drives correct
routing decisions across different LLMs.

Same prompt. Same fixtures. Different models.

## Fixtures

Test cases live in `fixtures.json`. Each scenario defines:

- A set of agents with descriptions (same shape as `AgentRegistry.from_descriptions()`)
- A default agent
- Test cases with expected routing targets and difficulty tags

### Difficulty Tiers

- **Easy**: Unambiguous — one clearly correct agent
- **Hard**: Ambiguous — multiple agents could reasonably handle it (any listed agent is accepted)
- **Adversarial**: Prompt injection, gibberish, off-topic — should fall back to default

### Bring Your Own Fixtures

```bash
python evals/run_eval.py --fixtures my_fixtures.json
```

The fixture format matches the library's API directly. See `fixtures.json` for the schema.

## Models

| Key | Provider | Model | Type |
|-----|----------|-------|------|
| `gpt-4o-mini` | OpenAI | gpt-4o-mini | LLM |
| `claude-haiku` | Anthropic | claude-haiku-4-5-20251001 | LLM |
| `gemini-flash` | Google | gemini-2.5-flash | LLM |
| `faiss-openai` | FAISS + OpenAI | text-embedding-3-small | Embedding |

## Output

Raw results are written to `evals/results/` (gitignored). The report generator
produces a markdown file with accuracy, latency, cost, confusion matrices, and
a list of every misclassification.
